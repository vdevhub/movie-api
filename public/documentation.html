Movie API Documentation

API Intro

Movie API is a REST API for an application that interacts with a database that stores data about different movies.

Essential Features

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

Supported Requests

Business Logic URL HTTP Method Request Body Response Body

Return data about a genre (description) by name (e.g., “Thriller”) /movies/genres/[genreName] GET None A JSON object holding data about the requested genre
Return data about a director (bio, birth year, death year) by name /movies/director/[name] GET None A JSON object holding data about the requested director
Allow new users to register /users POST A JSON object holding data about the user to be added A JSON object holding data about the newly added user and their new ID
Allow users to update their user info (username) /users/[id] PUT A JSON object holding data about the user to be updated A JSON object holding data about the updated user
Allow users to add a movie to their list of favorites /users/[id]/[movieTitle] POST None A text message indicating whether the movie was successfully added to the user's favorite movies list
Allow users to remove a movie from their list of favorites /users/[id]/[movieTitle] DELETE None A text message indicating whether the movie was successfully removed from the user's favorite movies list
Allow existing users to deregister /users/[id] DELETE None A text message indicating whether the user with the specified ID was removed
